The Web Application is designed to detect Covid-19 & Viral Pneumonia

“I've designed this Web Application based on a Deep Convolutional Neural Network with the help of the Tensorflow & Django framework. It uses Chest X-Rays to detect covid with higher accuracy in real-time.”

Covid-19

The model quickly determines whether or not the chest x-ray has covid-19 patterns.

Viral Pneumonia

It can quickly detect whether the chest x-ray has pneumonia patterns or not and display the result.

Tensorflow Framework

TensorFlow is a free and open-source machine learning and artificial intelligence framework.

Fast & Accurate

The model's detection rate is 90% accurate, and the results are displayed in real-time on the screen.

Neural Netork

Deep Convolution Neural Network is used to train the model to detect covid & pneumonia patterns.
